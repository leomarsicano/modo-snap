# Dashboard Pessoal do Leo

Primeira base do dashboard pessoal em React + Vite.

## O que já tem
- Tela inicial com resumo do dia
- Cards de rotina, saúde e financeiro pessoal
- Bloco de pendências
- Visual escuro, limpo e executivo

## Rodar localmente
```bash
npm install
npm run dev
```
